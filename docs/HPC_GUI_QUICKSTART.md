# HPC/SLURM GUI Quick Start (Current)

## Purpose

The HPC tab is an Advanced editor for SLURM settings stored in project.json.
It is intended for power users only.

## Steps

1. Start the GUI:

```bash
python prism_app_runner.py
```

2. Load a project in Projects.
3. Open the HPC tab.
4. Expand Advanced: SLURM Settings.
5. Edit fields and Save Settings to Project.
6. Run from Run App.

## Notes

- No direct job submission from the HPC tab.
- SLURM settings are stored in project.json.
